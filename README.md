# IT3040 Assignment 1 - Playwright Automation Testing

## Student Registration Number
IT23319110

## Project Description
This project contains automated test scripts for testing the
SwiftTranslator Singlish to Sinhala conversion system using Playwright.

The automation covers:
- Positive Functional Test Cases (Pos_Fun_0001 to Pos_Fun_0024)
- Negative Functional Test Cases (Neg_Fun_0001 to Neg_Fun_0010)
- UI Test Cases (Pos_UI_0001)

All automated tests are based on the documented Excel test cases.

## Requirements
- Node.js v18 or higher
- npm
- Playwright

## Installation
npm install

## Run All Tests
npx playwright test

## View Test Report
npx playwright show-report

## Browsers Tested
- Chromium
- Firefox
- WebKit
